# Support

[한국어](SUPPORT.ko.md)

Quilla support is intentionally narrow for the 0.1.1 Windows pilot.

## Included

- First-run setup help for Windows x64 and Unreal Engine 5.5.
- Help interpreting `quilla-first-run.json` and Blueprint smoke artifacts.
- Guidance for package verification, manifest hash checks, and unsigned binary
  status.
- One installation or smoke-test follow-up for a paid pilot purchase.

## Not Included

- General Unreal project debugging unrelated to Quilla setup.
- Unreal Engine versions other than 5.5 unless explicitly agreed.
- Project build errors, Blueprint authoring work, or game-specific bug fixing.
- GUI debugger, profiler, crash dump analyzer, or IDE support.

## Support Channel

Use the public repository issue tracker for setup and package verification
questions. Do not post proprietary project source, private assets, secrets,
license keys, or unreleased game content. Share sanitized first-run or smoke
JSON excerpts instead.

For paid pilots, reply through the purchase or delivery thread with:

- Quilla package version and zip SHA-256.
- Unreal Engine version.
- First-run result JSON.
- Blueprint smoke artifact JSON, if available.
- A short description of what you expected and what happened.

## Refund Rule

If first-run setup fails on a Windows UE 5.5 project because of a Quilla product
issue and the issue cannot be resolved within 7 days, the purchase is eligible
for refund. Failures caused by unsupported Unreal versions, broken local Unreal
installations, project build errors, or incorrect Blueprint path/node inputs are
support issues, not automatic refund reasons.
